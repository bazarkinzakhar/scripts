набор инструментов для администрирования и автоматизации

homelab-docker — конфиги для мониторинга (prometheus + grafana) и веб-сервера
terraform-docker — пример iac для развертывания контейнеров
ansible-base — универсальный плейбук для установки docker на debian/ubuntu
network-tools — скрипт для массовой проверки доступности портов
backup — бэкап данных. запуск: -s [откуда] -d [куда] -t [дней хранения]
